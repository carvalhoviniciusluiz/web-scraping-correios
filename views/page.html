Web Scraping Correios

Acessando o webservice de CEP dos correios

Para acessar o webservice, um CEP no formato de {8} digitos deve ser fornecido, por exemplo: "01001000"

Também é possível informar um endereço, a página dos correios sugere não utilizar nº da casa/ apto/ lote/ prédio ou abreviatura.

Acessando o webservice de CEP

Exemplo de pesquisa por CEP: http://localhost:3000/v1/cep/68900075

Acessando o webservice por endereço

Exemplo de pesquisa por Endereço: http://localhost:3000/v1/cep/1852

Correios: Pesquisa de CEP

O formulário usado neste projeto para obter os dados de endereço devolvidos via api encontra se disponível no link https://buscacepinter.correios.com.br/app/endereco/index.php?t que também pode ser facilmente acessado realizando a consulta no formulário Busca CEP ou Endereço disponível na página dos correios.
